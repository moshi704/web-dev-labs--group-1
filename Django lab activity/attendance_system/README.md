#  Student Attendance Management System

##  Project Overview
The Student Attendance Management System is a web-based application developed using Python and Django framework. It is designed to simplify and digitize the process of recording and managing student attendance in academic institutions. The system helps in reducing manual errors, improving efficiency, and providing a structured way of handling academic attendance records.


##  Objectives
The main objectives of this project are:

- To develop a centralized system for managing student attendance
- To reduce manual record keeping and human errors
- To improve efficiency in attendance tracking and reporting
- To apply Django framework concepts in a real-world project
- To design and implement relational database structures
- To enable future scalability for advanced features such as reporting and analytics


##  What Has Been Done

The following tasks have been completed so far:

- Installed and configured Python development environment
- Installed Django framework and verified setup
- Created Django project: `attendance_system`
- Created Django app: `attendance`
- Configured installed apps in `settings.py`
- Set up URL routing for project and app
- Created a basic home view returning a response message
- Planned system database structure (ERD design)
- Identified main system entities:
  - Student
  - Course
  - Lecturer
  - Attendance
- Set up project structure for further development

##  Limitations

The current version of the system has the following limitations:

- No graphical user interface (UI) has been developed yet
- Database models are not fully implemented in Django models
- No authentication system for students or lecturers
- Attendance marking is not yet automated
- No reports or analytics features implemented
- System currently runs only in development mode
