Student Grade Calculator

Objective

To develop a web-based Student Grade Calculator that allows users to enter student scores, calculate grades automatically, and display results instantly using JavaScript DOM manipulation.

Features

- Accepts student scores as input.
- Validates scores to ensure they are between 0 and 100.
- Calculates grades (A, B, C, D, F) based on score ranges.
- Displays results dynamically without reloading the page.
- Clears the input field after calculation for new entries.

Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation)

What Was Done

- Designed a user-friendly interface.
- Implemented event handling using JavaScript.
- Added input validation.
- Developed grade calculation logic using if-else statements.
- Updated results dynamically on the webpage.
- Implemented input reset functionality.

Challenges Faced

- Handling invalid user inputs.
- Ensuring real-time DOM updates without page refresh.
- Maintaining a clean and responsive user interface.

Expected Outcome

The application enables lecturers and students to calculate and view grades instantly, improving efficiency and user experience.
