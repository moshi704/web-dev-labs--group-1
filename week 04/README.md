# Understanding Layout Control in CSS

## Objective
The goal of this lab is to develop hands-on control over how elements are positioned and spaced on a webpage using only core CSS layout tools with no Flexbox or CSS Grid was used at any point.

## What Was Done

### Part 1: Box Model
Three stacked boxes were created, each containing a heading and a paragraph and CSS properties padding, border and margin were applied to each box.

### Part 2: Positioning
A card container was built using position: relative, position: absolute. The badge sits at the top-right corner of the card, anchored to the card itself and not the page. A fixed header was added at the top of the page that stays visible during scroll. 

### Part 3: Float and Clear
A two-column layout was built using float: left on the main content section and float: right on the sidebar,and a footer. .

### Part 4: Integrated Layout
All three concepts were combined into a single page. 

## Challenges
* The float and clear behavior was confusing at first.
* Managing the fixed header and sticky nav at the same time.
* Keeping the layout from breaking on different screen sizes was difficult.
