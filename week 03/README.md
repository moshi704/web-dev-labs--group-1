# Website Project

## Objective

The objective of this project is to develop a website, incorporating HTML structure, CSS styling, and the integration of external media assets such as images.

## What Has Been Done

* **HTML Structure**: Created a structured webpage with a header, navigation bar, and distinct sections for "About Us," "Skills," and "Daily Routine".
* **Styling**:
* Implemented both internal CSS within the HTML file and an external stylesheet (`style.css`) to control the appearance of the website.
* Applied styling to navigation elements, including hover effects, and formatted headers and body text using specific fonts and colors.


* **Media Integration**: Included a Python code snippet image (`coding.jpg`) in the "About Us" section to demonstrate image embedding.


## Challenges

* Balancing the use of internal versus external CSS styles.
* Managing specific visual properties like border-bottom styling and color schemes to ensure readability and desired aesthetics.
