Objective
The objective of this project was to design and develop a functional web page using HTML and CSS.

This include;
Creating the webpage structure using HTML
Applying styling and layout design using CSS.
Organizing content into clear sections.
Using different CSS selectors.
Applying External CSS, Internal CSS, and Inline CSS.

Challenges Faced
Managing proper page layout and alignment
Styling elements consistently
Ensuring clean code structure
Understanding how HTML and CSS work together effectively
