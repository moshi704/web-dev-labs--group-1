
# Web Development Labs - Group 1

## Objective
The objective of this project is to create a multi-page website as part of our introductory web development coursework. The goal was to build a functional navigation structure, understand page linking, and integrate local and external media assets.

## What Has Been Done
Successfully developed and linked a series of HTML pages to form a cohesive website:
* **`index.html`**: The home page of our website
* **`about.html`**: A page providing information about our group 
* **`contact.html`**: A page containing our contact information 
* **`external.html`**: A page showcasing both a locally hosted image (`image.jpg`) and an externally linked image [cite: external.html].
* **Navigation**: Each page includes a navigation bar to allow users to move between the Home, About, Contact, and External pages [cite: index.html, about.html, contact.html, external.html].

## Challenges
Ensuring that every page contained a complete and working navigation bar required careful attention to detail.
Managing the paths for both locally stored images and remote online image sources was a bit difficult.
understanding how browsers interpret file paths and URLs.
