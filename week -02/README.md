# Web Development Student Portal Project

## Objective

The objective of this project is to build a foundational student portal website, focusing on structuring web content using HTML, implementing navigation between pages, and organizing information through forms and tables.

## What Has Been Done

* **Home Page**: Developed `index.html` to serve as the portal entry point, featuring a navigation menu and organized lists for daily routines and hobbies.
* **Contact Management**: Created `contact.html` containing two distinct forms: a basic contact form and a structured registration form that utilizes text inputs, radio buttons for gender selection, and checkboxes for course selection.
* **Timetable Representation**: Implemented `timetable.html` using an HTML table to display a structured weekly schedule, incorporating row spanning and column spanning to manage complex layout requirements.
* **Navigation**: Established links between the main portal page, the contact page, and the timetable page to create a functional, multi-page website navigation flow.

## Challenges

* Managing table layout complexity, specifically using `rowspan` and `colspan`.
* Structuring the registration form within a table.
